# Bug reports

## 1. Не работает фильтр цены "До"

- Описание:
В результат поиска включены объявления, не удовлетворяющие условиям фильтра верхней границы цены.

- Шаги воспроизведения:
    1. Поля ввода диапазона цены "От" и "До" пустые
    2. В поле фильтра цены "До" ввести "1000".

- Фактический результат:
В списке результатов есть объявления с ценой выше 1000, указанного в значении фильтра цена "До"

- Ожидаемый результат
Показаны все объявления, удовлетворяющие условиям цены до 1000 рублей включительно

- Платформы: десктоп
- Приоритет: Критический

## Урезание списка результатов при вводе фильтра цены "От"

- Описание: при вводе значения "0" в поле фильтра цены "От" список результатов меньше, чем при пустом значении поля.

- Шаги воспроизведения:
    1. В поле фильтра цены "До" ввести число "1000"
    2. В поле фильтра цены "От" ввести число "0"
- Ожидаемый результат:
Список результатов на шаге 1 и шаге 2 одинаковый.

- Фактический результат: объявлений после ввода фильтра "От 0" меньше (1 объявление), чем при пустом поле "От" (4 объявления).
- Платформы: десктоп
- Приоритет: Критический

## Кнопка запуска таймера не работает

- Описание: после остановки таймера, кнопка запуска не активирует таймер заново

- Шаги воспроизведения:
    1. Нажать кнопку остановки таймера
    2. Нажать кнопку запуска таймера
- Ожидаемый результат:
Таймер был остановлен и снова запущен.

- Фактический результат: таймер переключился в состоянии остановки, завис. При нажатии на кнопки "обновить" и запуска таймера - ничего не происходит
- Платформы: десктоп
- Приоритет: Критический

## Страница статистики недоступна после обновления

- Описание: после обновления страницы, она становится недоступной.

- Шаги воспроизведения:
    1. Открыть главную страницу
    2. Перейти на страницу "Статистика"
    3. Обновить страницу
- Ожидаемый результат:
Страница обновлена, доступна для взаимодействия

- Фактический результат: чёрный экран и сообщение об ошибке "Page not Found"
- Платформы: десктоп
- Приоритет: Блокирующий

## Таймер останавливается, если страница не активна

- Описание: после запуска таймера обновления статистики он останавливается, если фокус пользователя переходит вне страницы.

- Шаги воспроизведения:
    1. Запустить таймер обновления статистики
    2. Перенести фокус пользователя браузера вне страницы статистики (перейти на другую страницу, свернуть браузер)
- Ожидаемый результат:
Таймер продолжает отсчет

- Фактический результат: таймер останавливается
- Платформы: десктоп
- Приоритет: Высокий
