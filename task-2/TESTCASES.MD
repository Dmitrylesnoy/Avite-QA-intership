# Test cases

## На десктопной версии сайта

### Главная страница

#### Проверить работу фильтра «Диапазон цен»

1. ID: FILTER-PRICE-01
2. Название: Проверка работы фильтра "Диапазон цен"
3. Методика: граничные значения
4. Предусловия:
    - Десктоп версия сайта
    - В системе существуют больше одного объявления с ценой до 10, от 10 до 1000 (включительно), от 1000 рублей
    - Поля ввода диапазона цены "От" и "До" пустые
5. Шаги:
    1. В поле фильтра цены "До" ввести "1000".
    2. В поле фильтра цена "От" ввести "10".
    3. Повторить шаги 1, 2 со значениями из таблица вариантов проверки
6. Ожидаемый результат:
    1. В списке результатов отображаются объявления, чья цена до 1000 рублей включительно. Среди них есть объявления удовлетворяющие условию цены от 10 рублей.
    2. В списке результатов отображаются только те объявления, чья цена от 10 до 1000 рублей включительно.

Таблица вариантов проверки:

|   | "От" | "До" | Результат |
|---|------|------|-----------|
| 1 | | 0  | Только объявления с ценой 0 |
| 2 | | -1 | Фильтр не применился |
| 3 | 0.1  | | Объявления с ценой от 0.1 |
| 4 | 0 |  | Показаны все объявления |
| 5 | -1 | | Фильтр не применился |
| 6 | | 0.1 | Объявления с ценой до 0.1 |
| 7 | 1000 | 10 | Объявления не найдены |
| 8 | 10 | 10000 | Только объявления с ценой от 10 до 10000 включительно |


#### Проверить сортировку «По цене»

1. ID: SORT-PRICE-01
2. Название: Проверка сортировки по цене
3. Предусловия:
    - Десктоп версия сайта
    - Существует больше 1 объявления с различными ценами.
4. Шаги:
    1. Выбрать сортировку по цене в меню сортировки
    2. Выбрать порядок "По возрастанию" в меню порядка
    3. Выбрать порядок "По убыванию" в меню порядка
5. Ожидаемый результат:
    1. В меню сортировки отображается "Цене",
    2. В меню порядка отображается "По возрастанию". Список объявлений упорядочен по возрастанию цены.
    3. В меню порядка отображается "По убыванию". Список объявлений упорядочен по убыванию цены.

#### Проверить работу фильтра «Категория»

1. ID: FILTER-CATEG-01
2. Описание: проверка фильтра категорий объявлений
3. Предусловия:
    - Выбран фильтр по умолчанию "Все категории"
    - Существует хотя бы 1 объявление каждой доступной категории
4. Шаги:
    1. Поочередно выбирать значение фильтра категории в соответствии с таблицей категорий.
5. Ожидаемый результат:
    1. В списке результатов отображаются только объявления, чьи категории соответствуют ожидаемым с таблицей категорий.

Таблица категорий:

| № | Категория | Ожидаемые категории в результатах |
|---|---|---|
| 1 | Все категории (default) | Отображаются объявления из всех доступных категорий |
| 2 | Любая категория из списка: Электроника, Недвижимость, Транспорт, Работа, Услуги, Животные, Детское, Мода | Каждое объявления из списка результатов относится к выбранной в фильтре категории |

#### Проверить работу тогла «Только срочные»

1. ID: FILTER-TGL-IMM-01
2. Описание: проверка применения фильтра приоритета "Срочные" при взаимодействии с тоглом "Только срочные"
3. Предусловия:
    - Фильтр приоритета выбран значением по умолчанию ("Все")
    - Тогл "Только срочные" выключен
4. Шаги:
    1. Активировать тогл "Только срочные"
    2. Деактивировать тогл "Только срочные"
    3. В фильтре приоритета выбрать "Срочный"
5. Ожидаемый результат:
    1. Тогл активирован, в поле фильтра приоритета выбрано "Срочный". В результатах поиска только объявления с приоритетом "Срочный"
    2. Тогл деактивирован, в поле фильтра приоритета выбрано значение по умолчанию ("Все"). В результатах поиска объявления всех приоритетов
    3. Тогл активирован, в поле фильтра приоритета выбрано "Срочный". В результатах поиска только объявления с приоритетом "Срочный"

### Страница Статистики

#### Протестировать контейнер управления таймером обновления статистики. (Работает кнопка «Обновить». Работает кнопка остановки таймера. Работает кнопка запуска таймера)

1. ID: STATS-TIMER-01
2. Описание: проверка управления таймера обновления статистики.
3. Предусловия:
    - Таймер запущен
4. Шаги:
    1. Нажать на кнопку "Обновить"
    2. Нажать на кнопку остановки таймера
    3. Нажать на кнопку запуска таймера
5. Ожидаемый результат:
    1. Значение времени таймера обновилось до 5 минут 0 секунд. Прогресс бар обнулился и запустился.
    2. Кнопка остановки таймера заменилась на кнопку запуска. Вместо прогресс бара появилось сообщение "Автообновление отключено"
    3. Кнопка запуска таймера заменилась на кнопку остановки. Прогресс бар обнулился и запустился.

## На мобильной версии сайта

- Проверить переключение между тёмной и светлой темой.

1. ID: THEME-TGL-01
2. Описание: проверка изменения отображаемого стиля вёрстки сайта, при переключении тогла темы.
3. Предусловия:
    - Включена тёмная тема, кнопка тогла с надписью "Светлая"
4. Шаги:
    1. Нажать на кнопку тогла темы
    2. Нажать на кнопку тогла темы
5. Ожидаемый результат:
    1. Применился стиль светлой темы (data-theme="light"). Надпись кнопки тогла "Темная".
    2. Применился стиль темной темы (data-theme="black"). Надпись кнопки тогла "Светлая"
