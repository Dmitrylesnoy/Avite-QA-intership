# Avito QA Task 2

## Быстрый запуск тестов

1. Установите зависимости:

```bash
npm install
```

2. Запустите тесты:

```bash
npm test
```

> `npm test` вызывает `npx playwright test`, поэтому дополнительных ручных настроек окружения обычно не требуется.

3. Отчет

По завершению тестирования отчет откроется автоматически. Отчет будет сохранён в директории playwright-results.
