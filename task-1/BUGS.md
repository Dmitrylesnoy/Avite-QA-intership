# Задание 1: Скриншот с багами

Найдите баги на приложенном скриншоте, приоритезируйте их и перечислите списком. Приоритеты: P0 — критический, P1 — высокий, P2 — средний, P3 — низкий.  
Пример: _P0 — не открывается объявление из выдачи._

![](bags.png)

## P0 - критический

- Сообщение "Ошибка, обновите страницу" никак не выделено для пользователя, при этом лента объявлений доступна для просмотра.

## P1 - высокий

- В результатах поиска содержится объявление, неудовлетворяющее поисковому запросу о месте. Есть объявление с локацией "Новосибирск", но запрос поиска "Котята с Москве"

- Тумблер сортировки объявлений по месту называется не в соответствии с поисковым запросом и местоположением пользователя. Название тумблера "Сначала из Можайска", местоположение из поискового запроса "... в Москве"

- Фильтр цены не применяется. В результатах поиска есть объявления с ценой ниже и выше указанного в фильтре диапазона цены.

- В результатах поиска содержится объявление, соответствующее введённому фильтру породы. (Объявление "Экзотический короткошерстный котёнок" с применённым фильтром Порода = Персидская)

- В результатах поиска содержится объявление, неудовлетворяющее введёному фильтру пола. (Фильтр пол = мальчик, но в ленте результатов есть объявление с полом = девочка)

- Счетчик объявлений в заголовке поиска показывает 19 - неверное количество объявлений списка (верное - 8; счетчик на кнопке применения фильтров тоже показывает 8).

## P2 - средний

- При наведении на кнопку со значком избранного/сравнения, показывается неверная подсказка "Не показывать мне похожие объявления"

- В цепочке категорий неверный порядок. Родительская категория "Кошки" находится внутри подкатегории "Персидская". ("Главная-животные-персидская-кошки")

- В результатах поиска содержится объявление, неудовлетворяющее категории объявлений. Содержится объявление с параметром "плюшевый" - относится к категории игрушек, но объявление отобразилось в категории "животные".

- Текст сообщения об ошибке написан мелким нечитаемым шрифтом

- Фильтр возраста содержит вариант "Щенок", хотя фильтры применяются к категории кошек

- В основном поле ввода поискового запроса не отображается введёный пользователем текст поиска.

- ТРЕБУЕТСЯ УТОЧНЕНИЕ ТРЕБОВАНИЙ РЕАЛИЗАЦИИ: Невозможно выбрать больше 1 породы в фильтр (как и что именно должно быть?)

- Нет кнопки возврата к "Все категории" (рядом с кнопкой возврата к категории "Животные")

## P3 - низкий

- Дублирование кнопки сброса породы. 1 - синяя кнопка "Сбросить" около заголовка фильтра "Порода", 2 - кнопка крестика у поля названия породы.

- У объявления, содержащего значек стрелки вниз, около цены в карточке объявлений не указано, насколько цена была снижена. (Уточнить конкретное ожидаемое поведение)

- Неодинаковое расстояние по горизонтали между карточками объявлений на 2 строке плиток результатов
